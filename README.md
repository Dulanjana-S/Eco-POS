# Eco-POS - POS System🛒

# A simple, offline Point of Sale (POS) system for small grocery stores to manage sales transactions efficiently.

## 🚀 Features

💡 Features / Key Functionality:
🛒 Product selection via 
🔢 Quantity entry and price display
➕ Add-to-cart functionality
🧾 Real-time cart view 
💰 Total calculation
💽 Database integration (SQL Server)
🔐 Local desktop application (no internet required)
Add Product, Generate Invoice, Sales History and more.

## ⚙️ Technology Stack
C# (.NET Framework)
Windows Forms (WinForms)
SQL Server 
ADO.NET (with `Microsoft.Data.SqlClient`)

## 📦 Prerequisites
Visual Studio 2022 or later
NET Framework (4.7.2+ recommended)
SQL Server Express or LocalDB

## ⚙️ Setup Instructions
1. **Clone the repository:**
 ```bash 
git clone https://github.com/yourusername/EcoGroceryMart.git
cd EcoGroceryMart
2. Open the solution in Visual Studio
3. Update the Connection String (if needed)
Inside Sales.cs:
private readonly string connectionString = "Data Source=YOUR_SERVER;Initial Catalog=EcoMart_POS;Integrated Security=True";
Replace YOUR_SERVER with your actual SQL Server name (e.g., DESKTOP-XYZ\\SQLEXPRESS).

▶️ Run the App
Press F5 or click Start in Visual Studio.
The form will load, allowing you to select products, enter quantities, and manage sales.
